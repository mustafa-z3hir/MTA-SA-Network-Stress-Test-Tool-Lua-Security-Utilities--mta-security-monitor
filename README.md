# MTA:SA Security & CPU Monitor

Server performance and security monitoring tool for MTA:SA.

## Features
- CPU spike detection
- Network diagnostics
- Alert history
- Admin commands

## Commands
| Command | Permission | Description |
|---------|------------|-------------|
| /checkcpu | Admin | Show CPU report |
| /clearcpu | Admin | Clear history |

## Installation
1. Copy folder to `resources/`
2. Run `/start mta-security-monitor`
